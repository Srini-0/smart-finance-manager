## About Kiro

Kiro is an AI-powered development assistant that helps with:
- Spec-driven development
- Code generation and refactoring
- Testing and quality assurance
- Documentation generation

## Project Context

**Kiro Finance** is a smart personal finance management application built with modern web technologies. This project demonstrates best practices in:
- React + TypeScript development
- Component-based architecture
- Local-first data storage
- AI-powered features
- Responsive design with Tailwind CSS

## Getting Started

To work with this project using Kiro:

1. Open the project in your Kiro-enabled IDE
2. Review the specs in `.kiro/specs/` for feature documentation
3. Check `.kiro/steering/` for coding standards and guidelines
4. Use Kiro's AI assistance for development tasks
